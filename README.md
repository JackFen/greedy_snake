# 🐍 Greedy Snake
一个基于 **TypeScript** 的经典贪吃蛇小游戏。  
支持键盘方向键操作，界面简洁，适合学习 TypeScript 基础与面向对象编程思想。

---

## 🎮 游戏特点
- 使用键盘方向键控制蛇移动  
- 撞到食物即可加长蛇身并增加分数  
- 撞到墙壁或自己身体，游戏结束  
- 使用 TypeScript + Webpack + Less 构建

## 🚀 安装与运行

```bash
# 克隆项目
git clone https://github.com/JackFen/greedy_snake.git

# 进入目录
cd greedy_snake

# 安装依赖
npm install

# 启动开发环境
npm run dev

# 打包生产环境
npm run build
```
